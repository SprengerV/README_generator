## About The Project

This application is a CLI for generating datailed README.md files.



### Built With

<ul>
	<li>NodeJS</li>
	<li>NPM</li>
</ul>






<!-- GETTING STARTED -->
### Installation

First navigate to the directory the app is downloaded to. In the terminal, enter "npm install" to install the necessary packages. Then enter "node index.js" and the app will open.



<!-- USAGE EXAMPLES -->
## Usage

Use this application to generate README.md files quickly in markdown syntax. I'm using it right now!

## Testing
